# Encapsulación
## Definición
La encapsulación protege los datos de un objeto y obliga a modificar su estado mediante métodos controlados.
## Aplicación construida
Se creó una clase `CuentaBancaria` con atributos privados y métodos públicos para depositar, retirar y consultar el saldo.
## Aprendizaje
La clase controla sus reglas internas y evita que otros objetos cambien el saldo sin validación.

## Como ejecutar 
1. Abrir terminal en la carpeta src
2. Compilar con: javac *.java
3. Ejecutar con: java Main

## Salida esperada
en la consola debe aparecer el nombre del titular y el saldo
tambien se debe ver el saldo final despues de realizar retiros y depositos
