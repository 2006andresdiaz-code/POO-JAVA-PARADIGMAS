# Composición
## Definición
La composición es una relación todo-parte fuerte donde el objeto principal controla la creación y la existencia de sus
componentes.
## Aplicación construida
Se creó una `Casa` que administra internamente sus `Habitacion`.
## Aprendizaje
Las habitaciones forman parte esencial de la casa y su ciclo de vida depende de ella.
## Comandos de compilación y ejecución
cd src
javac *.java
java Main
