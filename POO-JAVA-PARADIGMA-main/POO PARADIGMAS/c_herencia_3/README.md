# Herencia
## Definición
La herencia permite crear clases hijas a partir de una clase padre para reutilizar atributos y métodos comunes.
Guía práctica • Repo POO-JAVA-PARADIGMAS
## Aplicación construida
Se creó una clase `Empleado` y dos clases derivadas: `Gerente` y `Desarrollador`.
## Aprendizaje
La jerarquía reduce repetición y organiza mejor el código.
## Como ejecutar 
1. Abrir terminal en la carpeta src
2. Compilar con: javac *.java
3. Ejecutar con: java Main
