# Abstracción

## Definición
La abstracción modela lo esencial de un objeto y deja los detalles concretos para las clases especializadas.

## Aplicación construida
Se creó una clase abstracta `Figura` y dos implementaciones: `Circulo` y `Rectangulo`.

## Aprendizaje
Primero se define la idea general y luego cada clase concreta implementa su comportamiento.

## Como ejecutar 
1. Abrir terminal en la carpeta src
2. Compilar con: javac *.java
3. Ejecutar con: java Main
