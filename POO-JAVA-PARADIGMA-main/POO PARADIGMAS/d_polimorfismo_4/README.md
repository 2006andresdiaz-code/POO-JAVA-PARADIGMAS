# Polimorfismo
## Definición
El polimorfismo permite usar una referencia común para trabajar con objetos distintos que responden de manera
diferente.
## Aplicación construida
Se usó la clase `Animal` y las clases `Perro` y `Gato`, cada una con su propia implementación de `hacerSonido()`.
## Aprendizaje
El mismo mensaje puede producir comportamientos distintos dependiendo del objeto real.
## Como ejecutar 
1. Abrir terminal en la carpeta src
2. Compilar con: javac *.java
3. Ejecutar con: java java Main
