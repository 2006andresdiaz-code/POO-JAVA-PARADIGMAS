# Agregación
## Definición
La agregación es una relación todo-parte débil donde los objetos contenidos pueden existir de manera independiente.
## Aplicación construida
Se creó un `Equipo` que agrupa objetos `Jugador`.
## Aprendizaje
Los jugadores pertenecen al equipo, pero no dependen totalmente de su existencia.
## Comandos de compilación y ejecución
cd src
javac *.java
java Main
