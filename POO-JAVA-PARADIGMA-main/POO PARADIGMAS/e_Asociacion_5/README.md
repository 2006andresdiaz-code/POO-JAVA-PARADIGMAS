# Asociación
## Definición
La asociación conecta objetos que colaboran entre sí, pero que pueden existir por separado.
## Aplicación construida
Se relacionó un `Profesor` con un `Curso`.
## Aprendizaje
La asociación representa una relación flexible entre clases
## Como ejecutar 
1. Abrir terminal en la carpeta src
2. Compilar con: javac *.java
3. Ejecutar con: java java Main
